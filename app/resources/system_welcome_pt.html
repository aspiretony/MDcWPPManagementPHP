Agora, você pode iniciar a criação das classes na estrutura do template:
  1. Crie a classe de controle. Ex: ProductForm, ProductList;
  2. Efetue login como administrador (admin), para registrar as permissões;
  3. Registre a classe como programa (Administração -> Programas). Você deve preencher o campo "classe de controle" com o nome da classe (Ex: ProductForm);
  4. Conceda permissões para a classe registrada para algum usuário ou grupo (Administration -> Users ou Administration -> Groups);
  5. Adicione o programa ao menu.xml, que contém a estrutura do menu;
  6. Realize um logout e um login com algum usuário com permissão para rodar o programa criado;
  7. Finalize a criação do programa.
