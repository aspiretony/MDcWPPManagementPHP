Ahora, usted puede iniciar la creación de las clases en la estructura del template:
  1. Cree la clase de control. Ej: ProductForm, ProductList;
  2. Ingrese con el login de administrador (admin), para registrar los permisos;
  3. Registre la clase como programa (Administración -> Programas). Usted debe completar el campo "clase de control" con el nombre de la clase (Ej: ProductForm);
  4. Conceda permisos para la clase registrada para algun usuário o grupo (Administración -> Usuarios o Administración -> Grupos);
  5. Agregue el programa al menu.xml, que contiene la estrutura del menu;
  6. Realize logout y un login con algun usuário con permisos para ejecutar el programa creado;
  7. Finalize la creación del programa.
